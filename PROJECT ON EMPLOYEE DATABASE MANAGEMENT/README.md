This Python project enables basic employee data management using MySQL database connectivity.  
It supports common database operations such as create, insert, update, delete, and display records.

## Features

- Connect to MySQL database
- Create database and table if not already present
- Insert new employee records
- Update existing employee details
- Delete employee records by ID
- Display all employee records
- Validate record ID before update/delete operations
