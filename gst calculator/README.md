# GST Calculator – Mini Python Project

This project demonstrates the basic calculation of GST (Goods and Services Tax) in India.  
It takes the **Cost Price (CP)** and applies two GST components:

- **CGST** — Central Goods and Services Tax
- **SGST** — State Goods and Services Tax

Finally, it displays the total amount after adding both taxes.

---

## Features

- Simple command-line based GST calculator
- Takes dynamic user input
- Calculates:
  - CGST amount
  - SGST amount
  - Final price including GST
- Easy to understand and modify

---

## Concepts Used

- Variables and arithmetic operations
- `float` input handling
- Basic GST calculation logic
- Python `input()` and `print()` functions
