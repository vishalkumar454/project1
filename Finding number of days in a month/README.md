# Number of Days in a Month – Mini Python Project

This Python program takes a month number (1–12) as input and displays the number of days in that month.  
It also handles leap year logic correctly when the selected month is February.

---

## Features

- Accepts month number from user  
- Checks if the entered month is valid  
- Uses leap year rules to calculate February days  
- Displays the total number of days in the selected month  

---

## Concepts Used

- Conditional statements (`if-elif-else`)
- User input using `input()`
- Type conversion using `int()`
- Leap year checking logic
